# 🎓 Udemy Clone

A simple Udemy clone website built using HTML and CSS. This project replicates the basic UI of Udemy with course listings, images, and hover effects.

---

## 🔥 Features

- 📚 Course listing section
- 🖼️ Image cards with hover effects
- 🎨 Clean and responsive UI
- 🔍 Navigation bar (basic)
- 💡 Beginner-friendly design
